# Update Existing Page

Suggested workflow:

1. Read the current content file and page implementation.
2. Preserve the content model unless there is a clear structural reason to change it.
3. Make the smallest durable update that improves the public site.
4. Re-check metadata, links, and visual consistency.
5. Capture new rules or decisions when the change affects future work.

Prioritize maintainable edits over one-off page tweaks.
