# Create Fragment

Create a new fragment in `content/fragments/`.

Requirements:

- keep the tone calm, premium, and visually led
- write with restraint; avoid generic startup language
- include frontmatter with `title`, `slug`, `date`, `excerpt`, `coverImage`, and optional `tags`
- keep the body concise and intentional
- prefer language that reads well both on the site and in social previews

Output should be a single Markdown file ready for review.
